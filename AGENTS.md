# AGENTS.md

## 目的
このファイルは AI エージェント共通の作業方針を定義します。

## 基本方針
- 会話言語: 日本語
- コメント言語: 日本語
- エラーメッセージ: 英語
- コミット規約: Conventional Commits (`type: description`)
- 日本語と英数字の間には半角スペースを挿入する

## 判断記録のルール
- 重要な意思決定を行う際は、判断内容、代替案、採用理由、前提条件を明示する。

## 開発手順
1. **プロジェクト理解**: 既存のコード構造、`package.json`、ドキュメントを確認する。
2. **環境構築**: `pnpm install` で依存関係を解決する。
3. **実装**: プロジェクトの命名規則やスタイルに従い変更を行う。
4. **検証**: `pnpm lint` を実行し、必要に応じて動作確認を行う。

## セキュリティ / 機密情報
- API キー、パスワード、パスなどの機密情報をコードに直接記述しない。
- Git に機密情報が含まれるファイルをコミットしない。
- ログに個人情報や認証情報を出力しない。

## リポジトリ固有事項
- このプロジェクトは VRCX の SQLite データベースを読み取って API として公開するものです。
- データベースへの書き込み操作は行わず、読み取り専用（readonly）で接続します。
- `src/environments.ts` で定義されている環境変数を使用して設定を管理します。
